# Hafiz Watches

A modern watch store website built with HTML, CSS, and JavaScript. Features a clean ui, firebase authentication, and admin panel.

## 🎠 Features

- Product browsing and display
- Firebase authentication
- Admin panel for managing products
- Mobile-responsive design
- Modern user interface

## 🖓 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Firebase
- Deployment: Vercel
